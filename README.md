# Projeto Fullstack - CRUD de Usuários

## Descrição
Aplicação fullstack desenvolvida com FastAPI no backend e React no frontend, permitindo o gerenciamento de usuários.

---

## Tecnologias

### Backend
- Python
- FastAPI

### Frontend
- React
- Material UI
- Recharts
- Framer Motion

---

## Funcionalidades

- Criar usuários
- Listar usuários
- Atualizar usuários
- Deletar usuários
- Visualização em gráfico
- Interface moderna com animações

---

## Integração

O frontend consome a API via requisições HTTP (fetch), trocando dados em JSON.
